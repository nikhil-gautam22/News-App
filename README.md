**NewsWave** is a responsive and dynamic News Application built using **ReactJS**, **Context API**, and **Axios**, powered by the [NewsAPI](https://newsapi.org/). It provides real-time news updates covering Indian and global headlines. Users can search for topics, filter by categories, and stay informed about trending personalities and events.

---

##  Features

- **Global & Indian News** coverage
-  **Search Functionality**
  - Search for news articles and Indian personalities
-  **Category Filtering**
  - Business, Entertainment, Health, Science, Sports, Technology, etc.
- **Real-time News Fetching**
  - Powered by NewsAPI's `/everything` endpoint
-  **Fast & Responsive UI**
  - Optimized for all screen sizes (mobile, tablet, desktop)
- **State Management**
  - Managed efficiently using React Context API
-  **Auto-Refresh on Search/Filter Changes**

---

##  Tech Stack

- **Frontend**: ReactJS, HTML, CSS, JavaScript
- **State Management**: React Context API
- **API**: [NewsAPI](https://newsapi.org/)
